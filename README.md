# ParaBank Selenium Automation Framework

This project is a UI automation testing framework built using:

Python
Selenium WebDriver
PyTest
Page Object Model (POM)

Website tested:
https://parabank.parasoft.com

---

## Test Scenarios Covered

User Registration

Login validation

Logout functionality

Transfer Funds functionality

---

## Project Structure

pages
    base_page.py
    home_page.py
    register_page.py
    transfer_page.py

tests
    test_register.py
    test_login.py
    test_logout.py
    test_transfer_funds.py

conftest.py

requirements.txt

---

## How to run tests

Install dependencies:

pip install -r requirements.txt

Run tests:

pytest -v
